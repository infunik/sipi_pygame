# Adventure Runner

Adventure Runner — это простая 2D-игра в жанре "избегай и собирай" с видом сверху, разработанная с использованием Python и библиотеки Pygame. Управляйте синим квадратом, собирайте желтые монеты, чтобы увеличить счет и размер, и уклоняйтесь от красных треугольников! В игре есть система возрастающей сложности, и она сохраняет ваш лучший результат локально с помощью SQLite.

## Особенности

*   Перемещение игрока в 4 направлениях (вид сверху).
*   Случайным образом появляющиеся препятствия (красные треугольники), которых нужно избегать.
*   Случайным образом появляющиеся бонусы (желтые монеты), которые нужно собирать.
*   Счет увеличивается при сборе бонусов.
*   Игрок немного увеличивается в размере при сборе бонусов (до максимального размера).
*   Препятствия случайным образом исчезают и появляются вновь со временем, добавляя непредсказуемости.
*   Сложность автоматически увеличивается в зависимости от продолжительности игры, влияя на частоту появления объектов.
*   Сохранение лучшего результата (High Score) с использованием базы данных SQLite (`highscore.db`).
*   Простое главное меню (Начать/Выход) и экран "Игра Окончена" (Перезапустить) с взаимодействием через кнопки.
*   Интерфейс на экране, отображающий текущий счет, уровень сложности, время до следующего уровня и количество объектов.

## Требования

*   **Python:** Рекомендуется версия 3.8 или выше.
*   **Pygame:** Основная библиотека для игры.
*   **SQLite 3:** Используется для базы данных рекордов (обычно включена в стандартную установку Python).

## Установка и запуск

1.  **Клонируйте репозиторий (или скачайте файл `adventure_runner.py`):**
    ```bash
    git clone https://github.com/infunik/sipi_pygame.git # Замените вашим URL репозитория, если применимо
    cd AdventureRunner
    ```
    Или просто скачайте файл `adventure_runner.py` в папку на вашем компьютере.

2.  **Установите Pygame:**
    Откройте терминал или командную строку и выполните:
    ```bash
    pip install pygame
    ```
    *(В зависимости от вашей настройки Python, может потребоваться использовать `pip3` или `python -m pip`, например: `python -m pip install pygame`)*

3.  **Запустите игру:**
    Перейдите в папку, куда вы сохранили/клонировали файл, в вашем терминале и выполните:
    ```bash
    python adventure_runner.py
    ```
    *(Опять же, может потребоваться использовать `python3`)*

## Как играть

*   **Цель:** Продержаться как можно дольше, собирая желтые монеты для максимального увеличения счета. Избегайте столкновений с красными треугольниками.
*   **Управление:**
    *   **W / Стрелка Вверх:** Двигаться Вверх
    *   **S / Стрелка Вниз:** Двигаться Вниз
    *   **A / Стрелка Влево:** Двигаться Влево
    *   **D / Стрелка Вправо:** Двигаться Вправо
    *   **Мышь:** Взаимодействие с кнопками меню.
    *   **ESC:** Выйти из игры в любое время.
*   **Игровой процесс:**
    *   Используйте клавиши движения для управления синим квадратом (персонажем).
    *   Собирайте желтые круги (бонусы), чтобы увеличить счет на 10 очков и немного увеличить размер вашего квадрата.
    *   Избегайте касания красных треугольников (препятствий). Столкновение с препятствием приведет к **Окончанию Игры** (Game Over).
    *   Препятствия будут случайным образом исчезать и появляться в новых местах.
    *   Уровень сложности игры увеличивается каждые 30 секунд, что может влиять на частоту появления или исчезновения объектов.
    *   Ваш текущий счет, уровень сложности и количество объектов отображаются в левом верхнем углу экрана.

## Система рекордов

Игра автоматически сохраняет наивысший результат, достигнутый во время игры. Этот результат хранится локально в файле `highscore.db` в той же папке, что и скрипт игры. Лучший результат отображается в главном меню и на экране окончания игры.
